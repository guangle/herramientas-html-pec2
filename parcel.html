P arcel es un module bundler que destaca principalmente por su sencillez, ya que no requiere casi ninguna configuración para empezar a trabajar con él. Parcel nos da todas las funcionalidades de un module boundler: empaquetado, conversion de formato, servidor de desarrollo, minificación, etc.

Una de las características principales de Parcel es que hace su trabajo (empaquetado, minificación, transpilación, etc) bastante rápido. Esto nos permitirá ser más ágiles en el desarrollo. Otras características que tiene Parcel y que hacen la vida más fácil al desarrollador son: muestra errores comprensibles, y de una forma amigable, servidor de desarrollo, remplaza los modulos 'en caliente' conforme el programador los va modificando..

Si encuentra los ficheros de configuración apropiados, Parcel ejecuta automáticamente transformaciones a nuestro código sin necesidad de instalar o configurar ningún plugin adicional. Ejemplos:

  • Configurando .babelrc transpilaremos nuestro javascript con babel
  • .postcssrc nos permitirá usar PostCSS aplicar transformaciones a nuestros css

Cuando llegue el momento y nuestra aplicación este lista, Parcel nos permitirá preparar nuestra web para el entorno de producción, aplicando una serie de optimizaciones (configurables) como minificación, concatenación, cacheo, etc.

Parcel es una magnifica opción para empezar en el mundo de los module boundlers y usarlo en proyectos sencillos. Si quieres tener un control total sobre la configuración y además disponer de un 'ejecutor' de tareas más completo, quizás sea mejor opción Webpack

Es muy sencillo! con Parcel tienes todas las herramientas necesarias para crear un biolerplate moderno

Un poco de código

Nuestro primer proyecto parcel en 3 sencillos pasos 
npm install -g parcel-bundler
Instalar parcel es muy sencillo, basta con ejecutar este comando 

                <html>
                  <body>
                    <script src="./index.js">
                  </body>
                </html>

                console.log('hola mundo desde parcel..');

parcel index.html
