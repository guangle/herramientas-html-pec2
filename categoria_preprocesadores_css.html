Los preproceadores CSS son herramientas muy usadas en el desarrollo front-end cuya finalidad es ayudar al desarrollador a la construcción de ficheros CSS. Se pueden considerar como una ampliación del lenguaje CSS que consigue que el código sea más conciso, fácil de escribir y sobre todo fácil de leer.

Estas herramientas de preprocesamiento, 'amplian' el lenguaje css para que podamos usar características que no están inicialmente incluidas en el lenguaje: variables, funciones, anidamientos, etc. Aunque estas herramientas incluyan características nuevas, al final del procesamiento el resultado que obtendremos será siempre un CSS estándar perfectamente entendible por los navegadores Quiero saber más.

La principal ventaja que tenemos de usar este tipo de herramientas es que conseguimos un un código más organizado, mucho más fácil de escribir y de leer para otros desarrolladores del proyecto. Por ejemplo: si utilizamos un color verde en 47 elementos de nuestra hoja de estilos, con css 'normal' deberíamos escribir el código hexadecimal del verde en todas las clases y selectores donde queremos usar el color. Si de repente necesitamos cambiar la tonalidad de verde por otro código hexadecimal.. ¡Tendremos que hacer 47 cambios en el (o los) fichero css(s)! Esto únicamenete para una propiedad.. Si por el contrario definimos nuestro color verde en una variable, tenemos un punto único donde cambiar la tonalidad y el cambio se verá aplicado a todos los sitios donde se use esa variable.

Los procesadores css suelen ofrecer tambien mecánismos de generación/inclusión de módulos que nos facilitan mucho la reutilización del código CSS que vayamos creando. Como desventaja, se podría decir que la utilización de este tipo de frameworks supone un incremento de la complejidad, ya que los desarrolladores deben aprender usar el lenguaje y la herramienta y además se incrementa un poco el tiempo de construcción, ya que todos los ficheros scss, less deben ser procesados antes de servirse al navegador, que únicamente entiende css en lo referente a estilos.
